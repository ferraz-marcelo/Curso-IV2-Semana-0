MDL - Material Design Lite

Material Design Lite (MDL) é um framework criado pelo Google que permite adicionar aparência e comportamentos aos websites, deixando eles mais atraentes, consistentes e funcionais.
Os templates, componentes (badges, buttons, cards, etc..), fontes e ícones do MDL foram criados em CSS, JavaScript e HTML e são compatíveis com os principais navegadores web e mobile, e estão prontos para adicionarmos direto ao nosso projeto ou até mesmo customizar.
Nesse website vamos mostrar o funcionamento e aplicação de alguns componentes, como Badges, Buttons, Cards, List e Tooltips.

O primeiro passo para utilizar o MDL é adicionar o código abaixo no elemento head.
Neste caso, podemos alterar apenas a segunda linha, que é referente a customização das cores dos temas, que são divididas em primárias e secundárias, que você pode definir neste Link, e logo abaixo do círculo de cores, é gerado um novo código para vocÊ copiar e alterar o código "padrão". 

        
          <link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons">
          <link rel="stylesheet" href="https://code.getmdl.io/1.3.0/material.indigo-pink.min.css">
          <script defer src="https://code.getmdl.io/1.3.0/material.min.js"></script>

Badges

Badges é utilizado para aplicações onde os desenvolvedores precisam de um elemento de notificação na tela.
O Badges é um pequeno circulo com uma informação para chamar a atenção do usuário, como por exemplo nas redes sociais e aplicativos de conversas, que aparece um número dentro do círculo para avisar que tem novas mensagens ou notificições.
Também é utilizado em site de compras, normalmente no "carrinho de compras" indicando a quantidade de itens que já foi adicionado.

Tooltips

Tooltips é uma ferramenta composta por uma imagem, normalmente já bem esclarecida sobre qual ação ela é referente, e um texto, que aparece ao passar o mouse sobre sobre a imagem com informações adicionais.
